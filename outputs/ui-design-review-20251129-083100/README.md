## 🆘 常见问题

### Q: 能否跳过某些任务？
**A:** P0 任务必须全部完成，P1-P3 可根据实际情况选择。

### Q: AI 代码助手执行出错怎么办？
**A:**
1. 检查 TASKS.md 中的验收标准
2. 手动修正错误
3. 继续下一任务

### Q: 如何处理与现有代码的冲突？
**A:**
1. 优先保留现有业务逻辑
2. 只修改 UI 和样式相关代码
3. 有疑问时参考 REPORT.md 中的问题描述
