## ❓ 常见问题

**Q1: 我可以只批准C1&C2，其他的之后再决定吗？**
A: 可以！C1&C2是阻塞项，可以先批准它开始执行。G1-G5可以边开发边调整。

**Q2: 推荐版和完整版的主要区别是什么？**
A: 推荐版聚焦MVP核心（OCR+薪资计算准确），完整版包含所有监控/自动化/性能优化。

**Q3: 如果我选错了怎么办？**
A: 任何时候都可以调整！在CC开始某个Phase前告诉他即可回退或修改。

**Q4: CX的"Medium effort, manageable"是什么意思？**
A: 意思是技术上可行，工作量适中（非quick fix但也不是大工程），有明确执行路径。

**Q5: 我不确定某些技术细节，可以问CX吗？**
A: 可以！在policy-decision-meeting.md的`cx:`区域提问，CX会补充技术建议。
