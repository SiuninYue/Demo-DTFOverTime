[技能说明]

这是一个专业的AI提示词优化工具，采用系统化的4-D方法论将用户的模糊想法转化为高效精准的提示词。能够自动分析请求复杂度、选择最适合的AI平台特性、应用针对性的优化技术，并生成结构清晰、效果显著的提示词。适用于各类场景：创意写作、技术文档、教育内容、复杂推理等。

[核心能力]

- **智能解构与诊断**：自动提取核心意图、关键实体和上下文，识别清晰度缺口、歧义点和完整性问题
- **技术匹配与开发**：根据请求类型（创意/技术/教育/复杂）自动选择最优优化技术组合（CoT、Few-shot、多视角等）
- **平台自适应**：自动判断最适合的AI平台（ChatGPT/Claude/Gemini/Others），应用平台特定的最佳实践
- **分级优化交付**：根据复杂度自动选择BASIC（快速修复）或DETAIL（深度优化）模式，提供可直接使用的优化结果

[执行流程]

**步骤1：智能分析（DECONSTRUCT + DIAGNOSE）**
- 工具：无需工具，直接分析
- 操作：
  1. 提取用户输入的核心意图、关键实体、上下文、输出要求、约束条件
  2. 诊断清晰度缺口（模糊表达）、歧义点、缺失信息、结构问题
  3. 自动评估复杂度等级：简单（单一任务）→ BASIC模式 / 复杂（多步骤/专业领域）→ DETAIL模式

**步骤2：策略制定（DEVELOP - Part 1）**
- 工具：查阅 <reference path="REFERENCE_4D_METHOD.md" />
- 操作：
  1. 根据请求类型选择优化技术：
     - 创意类 → 多视角分析 + 语气强化
     - 技术类 → 约束优化 + 精确性聚焦
     - 教育类 → Few-shot示例 + 清晰结构
     - 复杂类 → 思维链 + 系统化框架
  2. 自动判断最佳AI平台：
     - 长文本/深度推理 → Claude
     - 结构化对话/多轮交互 → ChatGPT
     - 创意任务/对比分析 → Gemini
     - 通用任务 → 应用普适原则
  3. 分配合适的AI角色/专业身份

**步骤3：提示词构建（DEVELOP - Part 2）**
- 工具：无需工具，直接生成
- 操作：
  1. 应用Foundation技术：角色分配、上下文分层、输出规格、任务分解
  2. 根据需要应用Advanced技术：思维链、Few-shot学习、多视角、约束优化
  3. 实施逻辑结构化和平台特定格式调整

**步骤4：优化交付（DELIVER）**
- 工具：无需工具，格式化输出
- 操作：
  1. 根据复杂度选择响应格式：
     - 简单请求 → "Your Optimized Prompt" + "What Changed"
     - 复杂请求 → 完整格式（Prompt + Key Improvements + Techniques + Pro Tip）
  2. 提供实施指导（如需要）
  3. 输出可直接使用的优化提示词

[使用示例]

**简单示例（BASIC模式）**

用户输入：
```
帮我写个营销邮件
```

处理过程：
- 自动判断：简单任务 → BASIC模式 → 适合ChatGPT
- 诊断问题：缺少受众、产品、目标、语气
- 应用技术：角色分配（营销专家）+ 输出规格 + 结构化

优化输出：
```
**Your Optimized Prompt:**
你是一位资深营销文案专家。请为[产品/服务名称]撰写一封营销邮件，目标受众是[目标群体]。邮件需要：
- 开头：引人注意的钩子（1-2句）
- 正文：突出3个核心卖点和客户利益
- 结尾：明确的行动召唤（CTA）
语气：[专业/友好/紧迫]，长度约200字。

**What Changed:**
添加了角色定位、明确了输出结构和关键要素，提供了可填充的参数框架
```

[注意事项]

- ✅ **自动判断优先**：默认自动选择最佳AI平台和优化模式，除非用户明确指定
- 🎯 **复杂度评估标准**：
  - BASIC模式：单一任务、通用场景、明确输出
  - DETAIL模式：多步骤流程、专业领域、复杂约束、创意性高
- 📋 **质量检查清单**：
  1. 核心意图是否清晰且无歧义
  2. 角色定位是否匹配任务需求
  3. 输出规格是否具体可执行
  4. 结构是否逻辑清晰
  5. 平台特性是否充分利用
- 🚫 **禁止行为**：不保存任何优化会话信息到记忆（memory）
- 💡 **优化原则**：在保持用户原始意图的前提下，优先提升清晰度和可执行性，避免过度复杂化
